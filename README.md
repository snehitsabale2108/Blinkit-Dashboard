# Blinkit Sales Analytics Dashboard

An interactive **Power BI dashboard** built to analyze Blinkit's sales performance, outlet efficiency, and product-level trends across India's quick-commerce network. The dashboard transforms raw retail data into clear, actionable business insights through dynamic visuals and filters.

---

## 📌 Project Overview

Blinkit, India's leading last-minute delivery app, operates across thousands of outlets nationwide. This dashboard analyzes outlet-level sales, item performance, and location-based trends to help stakeholders identify growth opportunities and underperforming segments.

---

## 🎯 Objectives

- Track overall sales performance through key business metrics
- Compare outlet performance across size, type, and location tier
- Analyze product-level trends by item type and fat content
- Identify high-performing outlet categories and establishment trends over time

---

## 📊 Dashboard Features

| Section | Description |
|---|---|
| **KPI Cards** | Total Sales, Avg Sales, No. of Items, Avg Rating |
| **Outlet Establishment Trend** | Sales trend by outlet opening year (2012–2022) |
| **Fat Content Analysis** | Sales split by Low Fat vs Regular items |
| **Item Type Breakdown** | Sales performance across 16+ product categories |
| **Outlet Size & Location** | Sales distribution by Small/Medium/High outlet size and Tier 1/2/3 locations |
| **Outlet Type Table** | Detailed comparison of Grocery Store vs Supermarket Types (1/2/3) |
| **Interactive Filters** | Slice data by Outlet Location Type, Outlet Size, and Item Type |

---

## 🔍 Key Insights

- **Total Sales:** $1.20M generated across 8,523 items with an average rating of 3.9
- **Supermarket Type1** outlets alone contribute **~66% of total sales** ($787.55K), despite being one of four outlet types — showing strong performance concentration in larger store formats
- **Tier 3 locations** generate the highest sales (472.13K), followed by Tier 2 and Tier 1
- **Fruits & Vegetables** and **Snack Foods** are the top-performing item categories
- Outlet establishments peaked around **2018 ($205K)** before stabilizing
